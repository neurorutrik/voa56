# Сайт автошколы ВОА

Готовый статический сайт без сборки.

## Что внутри
- `index.html` — главная
- `training.html` — обучение
- `students.html` — курсантам
- `docs.html` — документация
- `management.html` — руководство
- `teachers.html` — педагогический состав
- `contacts.html` — контакты
- `style.css`, `script.js`
- `assets/files/` — документы
- `assets/img/` — фотографии и схемы

## Как разместить
1. Загрузите **все** файлы из этой папки на хостинг.
2. Стартовая страница — `index.html`.
3. Никакая сборка не нужна.

## Что отмечено отдельно
В документации некоторые пункты помечены как «Файл не приложен». Я их не выдумывал — для них в загруженных материалах не было однозначного файла.
